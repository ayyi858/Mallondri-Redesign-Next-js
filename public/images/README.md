# Gambar Website Mallondri

Letakkan gambar Anda di folder ini.

## File yang didukung

| Nama file   | Digunakan di   | Rekomendasi ukuran |
|------------|----------------|---------------------|
| `hero.jpg` atau `hero.webp` | Hero section (gambar utama) | Min. 1200Ã—800 px, landscape |

Jika **tidak** ada file `hero.jpg` / `hero.webp`, website akan memakai gambar placeholder dari Unsplash.

## Format

- **hero**: JPG atau WebP, rasio 4:3 atau 3:2 (landscape).
- Nama file harus persis: `hero.jpg` atau `hero.webp`.

Setelah menambah file, refresh halaman beranda untuk melihat gambar Anda.
